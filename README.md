# Language & Communication Rules

1. **Interaction Language**:
   - You (the Agent) must communicate, explain plans, and answer questions in **Portuguese (pt-BR)**.
   - I (the User) will provide instructions in Portuguese.

2. **Code Language**:
   - **ALL** code (variable names, function names, classes, database schemas) must be in **English**.
   - Code comments should be in **English** (unless specifically asked otherwise).
   - Commit messages should be in **English** (following Conventional Commits).
